# Vue 站点信息架构

所有用户可见站点均使用 Vue 3；文档站使用 VitePress。

## 1. 产品门户 `apps/portal`

- 首页：产品定位、核心优势、主要功能、典型场景和版本信息。
- 产品：核心 SDK、Vue 组件、空间分析、专题可视化和项目模板。
- 解决方案：自然资源、智慧城市、交通、环境、应急和园区。
- 开发生态：插件、模板、数据接入和工程工具。
- 更新：版本日志、路线图和兼容性。

## 2. 开发示例 `apps/examples`

- 多级分类、全文搜索、标签和能力筛选。
- 地图运行区、源码区、说明区、配置区和测试状态。
- Vue SFC、TypeScript 和 JSON 配置多标签展示。
- 支持新窗口运行、复制、下载、在线编辑和分享。
- 每个示例关联能力 ID、教程、API 和测试。

## 3. 开发文档 `apps/docs`

- 入门、核心概念、配置体系和 Vue 集成。
- 图层、控件、编辑、查询、分析、可视化和实时数据。
- 数据服务、请求管线、认证、安全、缓存和错误处理。
- API、FAQ、最佳实践、性能优化和工程规范。

## 4. 模板中心

- 基础地图浏览器。
- GIS 数据编辑工作台。
- 综合态势与指挥中心。
- 专题分析看板。
- 移动端地图。

所有站点共享设计令牌、导航、搜索索引、版本信息和示例元数据。
