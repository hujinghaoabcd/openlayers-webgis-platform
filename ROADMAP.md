# Roadmap

## Phase 0 — 工程基线（已完成）

- 建立 monorepo、核心 SDK、Vue 组件、产品门户、示例站和文档站。
- 建立统一类型、构建、测试、文档生成和持续集成流程。

## Phase 1 — 地图与图层核心

- 完成配置 Schema、地图视图、投影注册、底图管理和图层生命周期。
- 完成栅格、矢量、矢量瓦片、图像和实时图层能力。
- 完成图层树、加载状态、错误处理、资源释放和基础性能测试。

## Phase 2 — 控件、标绘与编辑

- 完成量测、绘制、编辑、捕捉、变换、切割、挖洞和撤销重做。
- 完成工具条、属性面板、上下文菜单、图例、书签和移动端交互。

## Phase 3 — 查询、服务与空间分析

- 完成统一请求管线、认证、中间件、缓存、取消和错误模型。
- 完成要素查询、属性查询、空间查询、数据编辑和批量事务。
- 完成客户端、Web Worker 和远程任务统一分析模型。
- 完成缓冲、叠加、裁剪、聚合、网络、地址和地形分析。

## Phase 4 — 专题可视化与实时能力

- 完成聚合、热力、图表符号、轨迹、流向、时间轴和 WebGL 专题渲染。
- 完成实时数据、打印、导出、录制、离线缓存和性能优化。

## Phase 5 — 示例、教程与 API 完整化

- 每项能力配套示例、教程、API、测试和变更记录。
- 示例站加入源码标签、在线编辑、新窗口运行、分享和下载。

## Phase 6 — 通用项目模板

- 完成基础地图、GIS 工作台、综合态势、专题门户和移动地图模板。
- 提供项目创建 CLI 和模板升级机制。
